Stock Trading Dashboard UI

⏳ Deadline: 48 Hours from start

The Objective

Your task is to build a responsive, single-page market overview for a modern web3 trading firm. The user needs to see a clean layout containing their total portfolio balance, a list of trending assets, and a data visualization area.

Technical Constraints

  • Allowed: Pure HTML, CSS (Flexbox/Grid), and Vanilla JavaScript.
  • Not Allowed: CSS frameworks like Tailwind or Bootstrap. No React or Vue. We want to see your raw fundamental skills.
  • Responsiveness: Your layout must adapt smoothly from a wide desktop monitor down to a mobile phone screen.
